# Todolist Simple Project

โปรเจกต์ Todolist แบบพื้นฐาน ฝึกทักษะการใช้ภาษา Javascript

---
## 🛠️ เครื่องมือที่ใช้พัฒนา (Technologies Used)

*   **HTML5:** โครงสร้างเว็บเว็บและการจัดเรียงเนื้อหา
*   **CSS3:** การตกแต่งหน้าตา, Responsive
*   **Javscript:** เพิ่ม event การกดแล้วแสดงผล เช่น onclick เพิ่มเพิ่มงาน , ลบงาน

---
## 🚀 วิธีการใช้งาน (Getting Started)

1.  **Clone โปรเจกต์นี้ไปยังเครื่องของคุณ:**
    ```bash
    git clone https://github.com/Testerzopa/Todolist.git
    ```
2.  **เปิดใช้งาน:**
    *   ดับเบิ้ลคลิกที่ไฟล์ `index.html` เพื่อเปิดใช้งานบนบราวเซอร์ได้ทันที
    *   หรือใช้ Extension เช่น **Live Server** บน VS Code เพื่อดูการเปลี่ยนแปลงแบบ Real-time
